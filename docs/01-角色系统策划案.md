# 角色系统策划案

## 1. 系统概述

角色系统是游戏的核心，管理玩家角色的属性、等级、职业、经验等基础数据。

## 2. 数据结构

### 2.1 角色基础数据 (PlayerData)

```typescript
interface PlayerData {
  // 基础信息
  id: string;                    // 角色唯一ID
  name: string;                  // 角色名称
  job: JobType;                  // 职业: 'warrior' | 'mage' | 'taoist'
  level: number;                 // 当前等级 (1-70)
  exp: number;                   // 当前经验值
  
  // 基础属性（不含装备加成）
  baseStats: BaseStats;
  
  // 当前状态
  currentHp: number;             // 当前生命值
  currentMp: number;             // 当前魔法值
  
  // 统计
  stats: PlayerStatistics;
  
  // 时间戳
  createdAt: number;             // 创建时间
  lastLoginAt: number;           // 最后登录时间
  lastOfflineRewardAt: number;   // 最后领取离线奖励时间
}

// 职业类型
type JobType = 'warrior' | 'mage' | 'taoist';

// 基础属性
interface BaseStats {
  maxHp: number;                 // 最大生命值
  maxMp: number;                 // 最大魔法值
  attack: number;                // 物理攻击
  magicAttack: number;           // 魔法攻击
  defense: number;               // 物理防御
  magicDefense: number;          // 魔法防御
  
  // 战斗属性
  critRate: number;              // 暴击率 (0-1)
  critDamage: number;            // 暴击伤害加成 (默认1.5)
  dodgeRate: number;             // 闪避率 (0-1)
  attackSpeed: number;           // 攻击速度 (每秒攻击次数)
  
  // 特殊属性
  hpRegen: number;               // 生命恢复 (每5秒)
  mpRegen: number;               // 魔法恢复 (每5秒)
  expBonus: number;              // 经验加成 (0-1)
  goldBonus: number;             // 金币加成 (0-1)
  magicFind: number;             // 掉宝率加成 (0-1)
}

// 玩家统计
interface PlayerStatistics {
  totalKills: number;            // 总击杀数
  totalBossKills: number;        // BOSS击杀数
  totalExpGained: number;        // 累计获得经验
  totalGoldEarned: number;       // 累计获得金币
  totalPlayTime: number;         // 总游戏时长(分钟)
  highestMap: number;            // 到达的最高地图ID
  firstKillBosses: string[];     // 首杀过的BOSS列表
}
```

### 2.3 等级经验配置

```typescript
const LEVEL_CONFIG = {
  maxLevel: 70,
  
  // 计算升级所需经验
  getExpToLevel: (level: number): number => {
    if (level <= 1) return 0;
    if (level <= 30) return Math.floor(100 * Math.pow(level, 1.6));
    if (level <= 50) return Math.floor(100 * Math.pow(level, 1.8));
    return Math.floor(100 * Math.pow(level, 2.0));
  },
  
  // 获取等级区间配置
  getLevelPhase: (level: number): LevelPhase => {
    if (level <= 20) return 'newbie';
    if (level <= 40) return 'mid';
    if (level <= 60) return 'advanced';
    return 'endgame';
  }
};

type LevelPhase = 'newbie' | 'mid' | 'advanced' | 'endgame';
```

## 3. 核心接口

### 3.1 角色管理接口

```typescript
class PlayerSystem {
  // 创建新角色
  static createPlayer(name: string, job: JobType): PlayerData;
  
  // 获取角色完整数据（含装备加成后的最终属性）
  static getPlayerData(playerId: string): PlayerData;
  
  // 获取角色最终战斗属性（基础+装备+生肖+BUFF）
  static getFinalStats(playerId: string): FinalStats;
  
  // 增加经验
  static addExp(playerId: string, exp: number): LevelUpResult;
  
  // 升级处理
  static levelUp(playerId: string): void;
  
  // 恢复生命/魔法
  static restoreHp(playerId: string, amount: number): void;
  static restoreMp(playerId: string, amount: number): void;
  
  // 消耗生命/魔法
  static consumeHp(playerId: string, amount: number): boolean;
  static consumeMp(playerId: string, amount: number): boolean;
  
  // 更新统计数据
  static updateStats(playerId: string, stats: Partial<PlayerStatistics>): void;
}

interface LevelUpResult {
  leveledUp: boolean;            // 是否升级
  newLevel?: number;             // 新等级
  levelsGained?: number;         // 升级层数
  rewards?: LevelUpReward;       // 升级奖励
}

interface LevelUpReward {
  hpIncreased: number;
  mpIncreased: number;
  attackIncreased: number;
  defenseIncreased: number;
  skillPoints?: number;          // 技能点（如有）
}

interface FinalStats extends BaseStats {
  // 最终属性包含所有加成
  damageMin: number;             // 最小伤害
  damageMax: number;             // 最大伤害
  dps: number;                   // 每秒伤害估算
}
```

### 3.2 属性计算逻辑

```typescript
class StatsCalculator {
  // 计算最终属性
  static calculateFinalStats(player: PlayerData): FinalStats {
    const base = player.baseStats;
    const equipment = EquipmentSystem.getEquipmentStats(player.id);
    const zodiac = ZodiacSystem.getZodiacStats(player.id);
    const setBonus = EquipmentSystem.getSetBonus(player.id);
    
    // 基础属性 = 角色基础 + 职业成长
    const final: FinalStats = {
      maxHp: base.maxHp + equipment.maxHp + zodiac.maxHp,
      maxMp: base.maxMp + equipment.maxMp + zodiac.maxMp,
      attack: base.attack + equipment.attack + zodiac.attack,
      magicAttack: base.magicAttack + equipment.magicAttack + zodiac.magicAttack,
      defense: base.defense + equipment.defense + zodiac.defense,
      magicDefense: base.magicDefense + equipment.magicDefense + zodiac.magicDefense,
      critRate: Math.min(0.5, base.critRate + equipment.critRate + zodiac.critRate),
      critDamage: base.critDamage + equipment.critDamage + zodiac.critDamage,
      dodgeRate: Math.min(0.3, base.dodgeRate + equipment.dodgeRate + zodiac.dodgeRate),
      attackSpeed: base.attackSpeed + equipment.attackSpeed + zodiac.attackSpeed,
      hpRegen: base.hpRegen + equipment.hpRegen + zodiac.hpRegen,
      mpRegen: base.mpRegen + equipment.mpRegen + zodiac.mpRegen,
      expBonus: base.expBonus + equipment.expBonus + zodiac.expBonus + setBonus.expBonus,
      goldBonus: base.goldBonus + equipment.goldBonus + zodiac.goldBonus + setBonus.goldBonus,
      magicFind: base.magicFind + equipment.magicFind + zodiac.magicFind,
      damageMin: 0,
      damageMax: 0,
      dps: 0
    };
    
    // 计算伤害范围
    final.damageMin = Math.floor(final.attack * 0.8);
    final.damageMax = Math.floor(final.attack * 1.2);
    
    // 计算DPS
    const avgDamage = (final.damageMin + final.damageMax) / 2;
    const critMultiplier = 1 + final.critRate * (final.critDamage - 1);
    final.dps = avgDamage * final.attackSpeed * critMultiplier;
    
    return final;
  }
  
  // 计算升级后属性
  static calculateLevelUpStats(job: JobType, newLevel: number): LevelUpReward {
    const config = JOB_CONFIG[job];
    return {
      hpIncreased: config.growth.hpPerLevel,
      mpIncreased: config.growth.mpPerLevel,
      attackIncreased: config.growth.attackPerLevel || config.growth.magicAttackPerLevel,
      defenseIncreased: config.growth.defensePerLevel
    };
  }
}
```

## 4. 自动恢复系统

```typescript
class RegenSystem {
  private static regenInterval: number = 5000; // 5秒恢复一次
  
  // 启动自动恢复
  static startRegen(playerId: string): void {
    setInterval(() => {
      const player = PlayerSystem.getPlayerData(playerId);
      const stats = PlayerSystem.getFinalStats(playerId);
      
      // 恢复生命
      if (player.currentHp < stats.maxHp) {
        const newHp = Math.min(stats.maxHp, player.currentHp + stats.hpRegen);
        PlayerSystem.restoreHp(playerId, newHp - player.currentHp);
      }
      
      // 恢复魔法
      if (player.currentMp < stats.maxMp) {
        const newMp = Math.min(stats.maxMp, player.currentMp + stats.mpRegen);
        PlayerSystem.restoreMp(playerId, newMp - player.currentMp);
      }
    }, this.regenInterval);
  }
}
```

## 7. AI开发提示

### 7.1 实现优先级

1. **P0 - 核心功能**
   - 角色创建与基础数据存储
   - 经验获取与升级逻辑
   - 属性计算（基础+装备）

2. **P1 - 重要功能**
   - 自动恢复系统
   - 统计数据追踪
   - 事件系统

3. **P2 - 扩展功能**
   - 成就系统对接
   - 排行榜数据

### 7.2 注意事项

- 所有数值计算使用整数，避免浮点误差
- 属性变化时触发事件通知UI更新
- 升级时自动恢复满血满蓝
- 最高等级70级，满级后经验不再增加
