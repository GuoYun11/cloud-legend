## 🔧 九、开发参考

### 9.1 CSS Variables 文件

已导出文件：`design-tokens.css`

包含所有设计Token，可直接在项目中引用。

### 9.2 快速引用

```html
<!-- 引入设计Token -->
<link rel="stylesheet" href="design-tokens.css">

<!-- 使用示例 -->
<div class="card" style="background: var(--color-surface-card);">
  <h2 style="color: var(--color-text-primary);">标题</h2>
  <p style="color: var(--color-text-secondary);">正文内容</p>
</div>
```

### 9.3 常用类名

```css
/* 背景 */
.bg-primary { background-color: var(--color-bg-primary); }
.bg-card { background-color: var(--color-surface-card); }

/* 文字颜色 */
.text-primary { color: var(--color-text-primary); }
.text-secondary { color: var(--color-text-secondary); }
.text-gold { color: var(--color-gold-primary); }

/* 品质色 */
.quality-normal { color: var(--color-quality-normal); }
.quality-rare { color: var(--color-quality-rare); }
.quality-legendary { color: var(--color-quality-legendary); }

/* 阴影 */
.shadow-card { box-shadow: var(--shadow-card); }
.shadow-glow-legendary { box-shadow: var(--shadow-glow-legendary); }
```
