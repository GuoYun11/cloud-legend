### 3.2 经验曲线调整

```javascript
// 新版经验曲线（匹配经典传奇节奏）
const EXP_CURVE = {
  // 升级经验公式
  expToLevel: (level) => {
    if (level <= 1) return 0;
    // 前30级较快，30-50级中等，50级后较慢
    if (level <= 30) return Math.floor(100 * Math.pow(level, 1.6));
    if (level <= 50) return Math.floor(100 * Math.pow(level, 1.8));
    return Math.floor(100 * Math.pow(level, 2.0));
  },
  
  // 挂机经验获取（每分钟）
  expPerMinute: (mapLevel, playerLevel) => {
    const baseExp = 15 * mapLevel;
    const levelDiff = playerLevel - mapLevel;
    
    // 等级压制：高于地图5级以上经验大幅减少
    if (levelDiff > 10) return Math.floor(baseExp * 0.1);
    if (levelDiff > 5) return Math.floor(baseExp * 0.5);
    if (levelDiff < -5) return Math.floor(baseExp * 1.2); // 挑战高级地图有加成
    
    return baseExp;
  }
};
```

### 8.2 Lv60 BOSS 战 DPS 模型（15s 窗口）

以下模拟假设：光芒/雷霆/烈焰套装 + 技能 Lv3

| 职业 | 普攻DPS | 技能DPS | 召唤/被动 | 爆发窗口DPS | 15s 总伤害 |
|------|---------|---------|-----------|-------------|------------|
| 战士 | 180 | 烈火+逐日 ≈ 320/次 | 护体（减伤） | 500+ | ≈6000 |
| 法师 | 120 | 火球+雷电+冰咆哮 ≈ 500/次 | 奥术+18%增伤 | 620+ | ≈7200 |
| 道士 | 100 | 火符+施毒 ≈ 200/次 | 召唤 150/s + 无极真气(+35% × 12s) | **爆发期 550+** | ≈6500 |

> **平衡结论**：
> - **战士**：稳定输出 + 高生存，DPS 居中；
> - **法师**：爆发最高，但血量最低，魔法盾消耗 MP 限制续航，高风险高收益；
> - **道士**：前期灵魂火符保底输出，后期无极真气+召唤体在 BOSS 战中形成 12s 爆发窗口，DPS 可逼近战法水平。无极真气 CD 期间依赖召唤和 DOT 维持输出，整体 DPS 略低于法师但高于战士持续阶段。
>
> 三职业 15s 总伤害比约 **战:法:道 = 1 : 1.2 : 1.08**，差距在合理范围内。
